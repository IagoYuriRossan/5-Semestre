# Aprendizagem de Máquina - 5º Semestre

Projeto da disciplina de Aprendizagem de Máquina.

## Bibliotecas Instaladas

- **numpy**: Computação numérica e manipulação de arrays
- **scikit-learn**: Algoritmos de machine learning
- **matplotlib**: Visualização de dados e gráficos

## Estrutura do Projeto

```
├── notebooks/     # Jupyter notebooks com experimentos
├── scripts/       # Scripts Python
├── data/          # Datasets e dados
├── models/        # Modelos salvos
└── requirements.txt
```

## Como usar

1. Ative o ambiente virtual:
   ```bash
   .venv\Scripts\activate
   ```

2. As bibliotecas já estão instaladas. Para instalar novamente:
   ```bash
   pip install -r requirements.txt
   ```

## Ambiente

- Python 3.14.2
- Ambiente virtual configurado em `.venv/`
