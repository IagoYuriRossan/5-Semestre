# Por onde começar?

Este documento é o ponto de entrada do projeto. Leia na ordem abaixo de acordo com seu objetivo.

---

## Quero só rodar o app

1. Abra o terminal
2. Execute:
```bash
cd "c:\Users\Alunos\Desktop\5-Semestre\Mobile II\atividade-mobile-ll"
npm install
npm start
```
3. Pressione `a` (Android) ou `w` (Web)
4. Teste as 3 abas: **Cadastro**, **Usuários**, **Admin**

> Na primeira execução, 12 usuários de exemplo são inseridos automaticamente no banco.
